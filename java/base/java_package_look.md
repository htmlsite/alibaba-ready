# Java

## 类型跑判断(个人说法)
- 后缀为able、Iterator 多数为接口
- 后缀为s多数为工具类

## 包结构

- java.lang.* 基础 重要
- java.math.* 精度算法
- java.awt.* 绘制图形、用户界面
- java.io.* 通过数据流、序列化和文件系统提供系统输入和输出
- java.nio.* 数据容器的缓冲区
- java.net.* 实现网络应用程序
- java.security.* 安全框架
- java.sql.* 问并处理存储在数据源 重要
- java.text.* 处理文本、日期、数字和消息 格式化和解析
- java.util.* 工具类  重要
